# 程式實作與系統模擬實作集
---

## 專案內容說明

### 1. **虛擬 CPU 指令解析與執行模擬 (Python)**
* 設計理念：為深入理解計算機組織中「指令週期 (Instruction Cycle)」之運作，實作一套基礎指令集並以 Python 實作硬體邏輯模擬。
* 實作技術：
    * 模擬通用暫存器 (Registers) 與標誌位元 (Flags) 之狀態轉換。
    * 基礎詞法解析，將自定義彙編語言轉換為可執行之指令流。
* 功能：支援 MOV, ADD, SUB, 以及條件跳轉 (JE) 等控制邏輯。



### 2. **數位 PWM 訊號產生與紀錄系統 (C++)**
* 設計理念：模擬數位輸出 (PWM) 如何透過控制佔空比 (Duty Cycle) 來達成類比訊號的效果。
* 實作技術：
    * 使用 std::thread 實作時序延遲，達成波形的動態顯示效果。
    * 利用 std::fstream 實作輸出重新導向，將模擬數據儲存為紀錄檔。
* 功能：展現數位訊號在實際硬體控制中的應用場景。

### 3. **情緒與睡眠健康評估系統 (C++)**
* 設計理念：應用資料結構中的線性容器，實作一個能追蹤健康趨勢並給予即時回饋的應用程式。
* 實作技術：
    * 採用 std::deque 維護 7 日動態視窗，進行多變量數據統計分析。
    * 實作年齡權重演算法，根據不同年齡層提供差異化的理想睡眠時數建議。
* 功能：整合情緒分數與睡眠品質指標，輸出綜合狀態的標籤與結果。

### 4. **RPG 遊戲框架實作 (Python)**
* 設計理念：以經典 RPG 戰鬥邏輯為主題，完整實作「物件導向程式設計 (OOP)」之核心觀念。
* 實作技術：
    * 類別封裝 (Encapsulation)：將角色屬性、裝備系統與戰鬥行為模組化。
    * 動態計算：透過類別方法即時計算受裝備加成後的最終攻擊與防禦數值。
* 功能：包含商店交易系統、隨機戰利品掉落以及武器鍛造強化邏輯。
